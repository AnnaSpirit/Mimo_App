Exercice 1

Créer une page simple :

Objectif : Créer une page HTML basique.

  1. Crée une page HTML avec :
    • Un titre de page (balise <title>).
    • Un en-tête <h1> avec un texte comme "Bienvenue sur ma page".
    • Un paragraphe <p> décrivant ce que tu apprends.
  2. Bonus CSS
    • Ajoute un style de base pour le texte (par exemple, change la couleur ou la taille)

Le titre de la page est : Exercices HTML & CSS

J'apprends le codage, est cette page combine le HTML : « HyperText Markup Language » et le CSS : « Cascading StyleSheets ».

Exercice 2

Liste et liens :

    Objectif : Ajouter des listes et des liens.

  1. Ajoute une liste à puces <ul> avec trois éléments (par exemple, tes hobbies ou des plats préférés)
  2. Ajoute une liste ordonnée <ol> pour une recette (3 étapes minimum).
  3. Insère un lien vers un site que tu aimes (ouvre-le dans un nouvel onglet avec l'attribut target="_blank").
  4. Bonus CSS
    • Mets une couleur différente pour les liens et supprime leur soulignement.

Mes passions :

  • Les animaux
  • Les arta créatifs
  • le code

Souffle-au-fromage :

  1. Etape 1: le four th.6 (180°C).
  2. Etape 2: Dans une casserole, faites un roux en faisant fondre le beurre et en y ajoutant la farine. Mélangez bien, jusqu'à ce que le beurre soit totalement absorbé. Ajoutez le lait en continuant à mélanger. Lorsque le mélange épaissit et commence à avoir la texture d'une béchamel, ajoutez le sel, le poivre et la muscade. Incorporez les jaunes d’œufs (gardez bien les blancs) l’un après l’autre, en tournant toujours, puis le gruyère râpé.
  3. Etape 3: Battez les blancs en neige très ferme. Ajoutez-les délicatement à votre préparation, au dernier moment.
  4. Etape 4: Beurrez et farinez le moule à soufflé (ou plusieurs petits) et remplissez-le jusqu'à moitié. Enfournez pendant 25 min environ sans ouvrir le four. Quand le soufflé gonfle et déborde légèrement du moule, servez aussitôt.

Exercice 3

Structure avancée

Objectif : Créer une mise en page avec des sections

  1. Crée une structure avec :
    • Une barre de navigation
      contenant trois liens (par exemple : Accueil, À propos, Contact).
    • Une section principale
      avec un article
      comprenant un titre et un paragraphe.
    • Un pied de page
      avec un copyright.
  2. Bonus CSS
    • Donne un arrière-plan différent pour chaque section pour les distinguer.

Exercice 6

Mini site responsif

Objectif : Créer une mise en page adaptative avec Flexbox ou Grid.

  1. Crée une page avec :
    • Une barre de navigation fixe en haut.
    • Une section principale avec deux colonnes (texte à gauche, image à droite)
    • Un pied de page collé en bas
  2. Utilise Flexbox ou CSS Grid pour organiser la mise en page.
  3. Ajoute un media query pour que la page s'adapte sur mobile (les colonnes passent en une seule).
  4. Bonus CSS
    • Ajoute une animation subtile sur un élément (par exemple, un bouton qui change de couleur lorsqu'on clique).
