Exercice 1

Créer une page simple :

Objectif : Créer une page HTML basique.

  1. Crée une page HTML avec :
    • Un titre de page (balise <title>).
    • Un en-tête <h1> avec un texte comme "Bienvenue sur ma page".
    • Un paragraphe <p> décrivant ce que tu apprends.
  2. Bonus CSS
    • Ajoute un style de base pour le texte (par exemple, change la couleur ou la taille)

Exercice 2

Liste et liens :

    Objectif : Ajouter des listes et des liens.

  1. Ajoute une liste à puces <ul> avec trois éléments (par exemple, tes hobbies ou des plats préférés)
  2. Ajoute une liste ordonnée <ol> pour une recette (3 étapes minimum).
  3. Insère un lien vers un site que tu aimes (ouvre-le dans un nouvel onglet avec l'attribut target="_blank").
  4. Bonus CSS
    • Mets une couleur différente pour les liens et supprime leur soulignement.

Exercice 3

Structure avancée

Objectif : Créer une mise en page avec des sections

  1. Crée une structure avec :
    • Une barre de navigation
      contenant trois liens (par exemple : Accueil, À propos, Contact).
    • Une section principale
      avec un article
      comprenant un titre et un paragraphe.
    • Un pied de page
      avec un copyright.
  2. Bonus CSS
    • Donne un arrière-plan différent pour chaque section pour les distinguer.
